# TypeScript

Repositorio correspondiente a los módulos 1 y 2.

## Contenido

- `modulo-1/`: inicialización de proyecto TypeScript, ejemplos básicos y laboratorio de lógica pura.
- `modulo-2/`: modelado de datos, uniones discriminadas, genéricos y simulación de acceso a datos.

## Estructura

- `modulo-1/src/`: código fuente del módulo 1
- `modulo-1/docs/`: documentación del módulo 1
- `modulo-2/src/`: código fuente del módulo 2
- `modulo-2/docs/`: documentación del módulo 2

## Tecnologías

- TypeScript
- tsx
- Node.js

## Ejecución

### Módulo 1
Entrar en la carpeta del módulo y ejecutar:

    npm install
    npm run dev

Comprobación de tipos:

    npx tsc --noEmit

Compilación:

    npm run build

### Módulo 2
Entrar en la carpeta del módulo y ejecutar:

    npm install
    npm run dev

Comprobación de tipos:

    npx tsc --noEmit

Compilación:

    npm run build

## Objetivo

Aplicar TypeScript estricto para:
- tipado de variables y funciones,
- modelado de entidades,
- uso de interfaces y type aliases,
- uniones discriminadas,
- genéricos,
- y reducción de errores en tiempo de ejecución.
